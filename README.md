# Quiz App

## Description
A simple **Quiz Application** built using **HTML**, **CSS**, and **JavaScript**.  
The app presents multiple-choice questions, allows users to select answers, tracks the score, and displays results at the end. Questions are randomly selected from a JSON dataset.

---

## Features
- Displays **multiple-choice questions**  
- Random question selection  
- Highlights **correct** and **incorrect** answers  
- **Next** button to proceed to the next question  
- Tracks and displays **score** at the end  
- Option to **play again**  

---

## How It Works
1. The app shows a random question with four possible answers.  
2. The user clicks an answer button.  
   - Correct answer: button turns green and score increments  
   - Incorrect answer: button turns red  
3. All buttons are disabled after selection.  
4. Click **Next** to go to the next question.  
5. After 10 questions, the total score is displayed, with an option to **restart the quiz**.

## How to Use
1. Open **index.html** in a browser.  
2. Read the question and click the correct answer.  
3. Click **Next** to continue through the quiz.  
4. View your total score at the end and choose to **Play Again** if desired.
